xml:space="preserve" bytes="6893"> {{Article for deletion/dated|page=Pydio|timestamp=20160920183547|year=2016|month=September|day=20|substed=yes|help=off}} {{Orphan|date=August 2016}} '''Pydio''' is an [[open source]] [[file sharing]] & synchronisation software that runs on the user's own [[Server (computing)|server]]{{Cite web|url=http://www.programmableweb.com/news/how-to-deploy-premise-file-sharing-and-sync-pydio/how-to/2016/08/19|title=Programmable Web : How to Deploy On-Premise File Sharing and Sync with Pydio|last=|first=|date=|website=|publisher=|access-date=}}{{Cite web|url=http://www.nextofwindows.com/top-four-self-host-cloud-storage-solution-alternative-to-dropbox|title=Top four self hosted cloud storage solutions alternative to Dropbox|last=|first=|date=|website=|publisher=|access-date=}} or in the [[Cloud computing|cloud]].{{Cite web|url=https://aws.amazon.com/marketplace/pp/B01H0O29U2/ref=sp_mpg_product_title?ie=UTF8&sr=0-2|title=Amazon Web Marketplace : Pydio|last=|first=|date=|website=|publisher=|access-date=}} == Presentation == The project was created by musician Charles Du jeu{{Cite web|url=http://www.silicon.fr/pydio-partage-les-fichiers-en-open-source-91481.html|title=Pydio partage les fichiers en open source|last=|first=|date=|website=|publisher=|access-date=}} (current [[Chief executive officer|CEO]] & [[Chief technology officer|CTO]]) in 2007 under the name AjaXplorer.{{Cite web|url=http://www.techrepublic.com/blog/tr-dojo/access-and-sync-folders-with-ajaxplorer/|title=Access and sync folders with AjaXplorer - TechRepublic|website=TechRepublic|access-date=2016-08-22}} The name was changed in 2013 and became Pydio (an acronym for ''Put Your Data in Orbit'').{{Cite web|url=https://blog.tfrichet.fr/ajaxplorer-devient-pydio-put-your-data-in-orbit-et-se-rapproche-downcloud/|title=AjaXplorer devient Pydio – Put Your Data In Orbit, et se rapproche d’ownCloud {{!}} Blog technophile tfrichet.fr|last=FRICHET|first=Thibaud|website=blog.tfrichet.fr|access-date=2016-08-22}}{{cite news|last1=van 't Ende|first1=Ben|title=The future of sharing: integrating Pydio and ownCloud|url=https://opensource.com/business/16/5/sharing-files-pydio-owncloud|accessdate=26 August 2016|publisher=Open Source|date=May 19, 2016}} Pydio runs on any server supporting a recent PHP version (PHP 5.4 recommended).{{Cite web|url=https://pydio.com/en/why-pydio/enterprise-distribution-requirements|title=Enterprise Distribution Requirements|date=2015-11-02|access-date=2016-08-22}} From a technical point of view, Pydio differs from solutions such as [[Google Drive]] or [[Dropbox (service)|Dropbox]].{{Cite web|url=https://www.unixmen.com/pydio-an-open-source-alternative-to-dropbox/|title=Pydio, an Open Source Dropbox Alternative|last=Cloud|first=La Lettre du|date=|website=|publisher=|language=fr-FR|access-date=2016-08-22}}{{Cite web|url=http://www.lemagit.fr/actualites/2240210420/Pydio-et-Red-Hat-ensemble-pour-attaquer-le-marche-francais-des-Box-dentreprises|title=Pydio et Red Hat, ensemble pour attaquer le marché français des " Box " d’entreprises|language=fr-FR|access-date=2016-08-22}} Pydio is not based on a public cloud,{{Cite web|url=http://www.lemondeinformatique.fr/actualites/lire-pydio-adapte-le-partage-de-fichiers-a-l-entreprise-59474.html|title=Pydio adapte le partage de fichiers à l'entreprise - Le Monde Informatique|access-date=2016-08-22}} the software indeed connects to the user's existing storages ([[Storage area network|SAN]] / Local FS, [[Samba (software)|SAMBA]] / [[Server Message Block|CIFS]], (s)[[FTP]], [[Network File System|NFS]], etc...) as well as to the existing user directories ([[LDAP]] / AD, [[SAML]], Raidus, [[Shibboleth (Internet2)|Shibboleth]]...), which allows companies to keep their data inside their legacy infrastructure, according to their data security policy{{Cite web|url=http://www.pcworld.com/article/2843932/french-startups-are-taking-a-more-privacy-centric-approach-file-sharing.html|title=French startups are taking a more privacy centric approach file sharing|last=|first=|date=2016-06-03|website=|publisher=|access-date=2016-08-22}} and user rights management. The software is built in a modular perspective; various plugins{{Cite web|url=https://pydio.com/en/docs/developer-guide/plugin-architecture|title=Plugin Architecture|date=2016-06-03|access-date=2016-08-22}} allow administrators to implement extra-features.{{Cite web|url=https://pydio.com/en/docs/references/plugins|title=Pydio Plugin List|last=|first=|date=|website=|publisher=|access-date=}} Pydio is available either through a community distribution, or an Enterprise Distribution.{{Cite web|url=http://www.toptechnews.com/article/index.php?story_id=033000KSKAWX|title=Pydio Brings Its Open Source File Sharing Service to Enterprise|last=|first=|date=|website=www.linformaticien.com|publisher=|access-date=2016-08-22}} == Features == * [[File sharing]] between different internal users and across other Pydio instances{{Cite web|url=https://pydio.com/en/community/releases/pydio-core/pydio-core-640-sharing-ux-performances|title=Pydio Core 6.4.0 - Sharing, UX, Performances|date=2016-03-29|access-date=2016-08-22}} * [[Desktop computer|Desktop]] [[Synchronization (computer science)|Synchronisation]] client{{Cite web|url=https://pydio.com/en/features/downloads/pydiosync-desktop-app|title=PydioSync Desktop App|date=2015-09-28|access-date=2016-08-22}} * iOS & Android mobile apps * [[Transport Layer Security|SSL/TLS]] Encryption * WebDAV File Server * Creation of dedicated Workspaces, for each line of business / project / client, with a dedicated user rights management for each workspace. * File Sharing with external users (private links, public links, password protection, Download limitation, etc.){{Cite web|url=http://www.demat-infos.com/info_article/m/302/pydio-ou-le-partage-securise-en-ligne.html|title=Pydio, ou le partage sécurisé en ligne|last=|first=|date=|website=|publisher=|access-date=}} * Online viewing and editing of documents with Collabora Office * Preview and editing of image files * Integrated Audio and Video reader == References == {{reflist|2}} == External links == * {{Official website|https://pydio.com/en}} * [https://pydio.com/fr/community Pydio Open Source Community] * [https://github.com/pydio Source Code on Github] [[Category:Free software]]
