Keyparc - Free Encryption for Everyone | homepage = [http://www.bloombase.com/ www.bloombase.com] }} '''Bloombase, Inc.''' is a company which develops and markets information security solutions to address data at-rest and in-flight threats. Bloombase launched its flagship Bloombase Security Server {{cite web | url=http://csrc.nist.gov/groups/STM/cavp/documents/dss/rsaval.html#496 | title=NIST RSA Algorithm Validation List}} supporting [[Advanced Encryption Standard|AES]],{{cite web | url=http://csrc.nist.gov/groups/STM/cavp/documents/aes/aesval.html#1041 | title=NIST Advanced Encryption Standard Algorithm Validation List}} [[Camellia (cipher)|Camellia]],{{cite web | url=http://info.isl.ntt.co.jp/crypt/eng/camellia/product.html | title=NTT Information Sharing Platform Laboratories Camellia Product Support }} [[SEED]] and [[ARIA (cipher)|ARIA]] cryptographic cipher algorithms attaining [[FIPS 140-2|NIST FIPS 140-2 validation]].{{cite web | url=http://csrc.nist.gov/groups/STM/cmvp/documents/140-1/1401vend.htm | title=NIST Validated FIPS 140-1 and FIPS 140-2 Cryptographic Modules}} Later, Bloombase announced StoreSafe [[virtual appliance]] edition for enterprise storage system security, supporting [[virtual machine]] containers including [[VMware ESX]], [[Citrix]] [[Xen]], [[Oracle Corporation|Oracle]] [[VirtualBox]], [[Microsoft]] [[Hyper-V]], [[IBM]] [[PowerVM]], and [[Red Hat]] [[Kernel-based Virtual Machine]] (KVM). Bloombase secures cloud storage and runs as compute instances on [[Amazon Web Services]], Microsoft [[Windows Azure]], [[Rackspace]], VMware [[VCloud Air|vCloud]], [[HP Helion]] and IBM [[SoftLayer]].
