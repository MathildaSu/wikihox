xml:space="preserve" bytes="11448"> {{Article for deletion/dated|page=FBReader|timestamp=20150703090453|year=2015|month=July|day=3|substed=yes|help=off}} {{Infobox software | logo = | screenshot = [[Image:Fbreader os2008.jpg|125px]] | caption = FBReader on [[Maemo (operating system)|Maemo]] | latest_preview_version = 2.0 beta 14 | latest_preview_date = {{Start date and age|2013|11|23}} | latest_release_version = 1.10.0.5 | latest_release_date = {{Start date and age|2014|04|21}} | programming language = [[C++]], [[Java (programming language)|Java]] | operating system = [[Cross-platform]] | language = [[Multilingual]] | genre = [[e-book]] reader | license = [[GNU General Public License|GPL]] ([[free software]]) | website = {{URL|fbreader.org}} }} '''FBReader''' is a [[free software|free]] and [[open source]] [[e-book reader]] for [[Linux]], [[Microsoft Windows]], and other platforms. It was originally written for the [[Sharp Zaurus]] and currently runs on many other [[mobile device]]s, like the [[Nokia Internet Tablet]]s, as well as [[desktop computer]]s. A preview of FBReaderJ (the [[Java (programming language)|Java]] port) for Google [[Android (operating system)|Android]] was released on April 13, 2008. Supported formats include [[EPUB]], [[FictionBook]], [[HTML]], [[plucker]], [[PalmDoc]], [[zTxt]], [[Text Compression for Reader|TCR]], [[Microsoft Compiled HTML Help|CHM]], [[Rich Text Format|RTF]], [[Open eBook|OEB]], [[mobipocket]] (mobi) without [[digital rights management|DRM]], and plain-text.[http://fbreader.org/about.php FBReader: About FBReader] [[File:FBReaderWindow.png|thumb|A desktop version of FBreader]] DAISY 3 format support was added to a fork of FBReaderJ for Android distributed by [[Benetech]] as Go Read for [[Bookshare]] on [[Google Play]]. == History == FBReader was originally released for the [[Sharp Zaurus]] in January 2005, a [[Maemo]] port was added in December 2005 for the [[Nokia 770]]. FBReader has since had binary packages released for many mobile device platforms and most major [[personal computer]] (PC) [[operating system]]s.[http://fbreader.org/obsolete/ FBReader: obsolete versions] The FBReader name with the [[FB]] prefix comes from FictionBook, an e-book format which is popular in [[Russia]], the country of FBReader's author.[http://talk.maemo.org/archive/index.php/t-555.html FBReader and its .fb2 format], maemo.org forum, 2005 The original FBReader was writting in C++, however in 2007{{Cite web|title = geometer/FBReaderJ|url = https://github.com/geometer/FBReaderJ/tree/23c429502a1ae00bde925b8fc9e17bde2a3ba5cf|website = GitHub|accessdate = 2015-07-03}} a fork called FBReaderJ was created, which was written in Java. As the Android platform became available in the following years, this fork became the codebase for the android app, which the C++ codebase remained in use for the other platforms.{{Cite web|title = geometer/FBReader|url = https://github.com/geometer/FBReader|website = GitHub|accessdate = 2015-07-03}} == Components == For easy [[cross-platform]] compiling, FBReader uses zlibrary, a cross-platform interface library. It allows recompiling for many platforms while disregarding the [[widget toolkit|GUI-toolkit]] used. ==Features== * support Multiple book [[tar]], [[zip (file format)|ZIP]], [[GZIP]] and [[BZIP2]] archives.http://wiki.mobileread.com/wiki/FBReader * encoding detection * generates contents table. * Embedded images * hyperlinks * Position indicator (substitutes for page number). * library building * Most Recent Book * last read positions for all previously opened books * List of last opened books. * Automatic hyphenations * Text search. * Full-screen mode. * Screen rotation by 90, 180 and 270 degrees. * Text-to-speech **To activate text to speech on the [[Android (operating system)|Android]] platform, install a TTS plugin, such as ''TTS+ plugin from Hyperionics''.[http://hyperionics.com/TtsPlus/ ''TTS+ Plugin'' - enhanced Text-To-Speech plugin for ''FBReader for Android''] == File format support == FBReader supports the following file formats:[https://fbreader.org/content/book-formats-supported-and-not-supported E-book formats, supported and not supported | Free eBook Reader - Free!] * [[EPUB]] : all the main features except the tables. CSS support is not full. * [[EPUB|EPUB3]] : does not support most of ePub 3 specific features * [[Mobipocket]] : opens non-encrypted *.mobi files. DRM-protected files are not supported. * [[FictionBook|FB 2.0]] : fully supported * [[FictionBook|FB 2.1]] : lacks support of tables * [[HTML]] : limited, sufficient support * Plain text : supported, might not correctly split text into paragraphs. * [[Rich Text Format|RTF]] : subset of RTF * [[Doc (computing)|DOC]] ([[Microsoft Word]]) : subset of DOC * [[Portable Document Format|PDF]] : **Android: via separate plugin with third-party library **Other platforms: not supported * [[DjVu]] : **Android: via separate plugin **Other platforms: not supported * [[Plucker]] : **Android: not supported at this moment **Other platforms: “absolute positioning” commands may be interpreted or ignored == Multi-Platform Support== * [[Microsoft Windows]] * [[BlackBerry 10]] * [[Mac OS X]] * [[FreeBSD]] * [[Linux]] * mobile [[Linux]] devices:http://wiki.mobileread.com/wiki/FBReader ** Sharp Zaurus with Qtopia-based ROMs, pdaXrom or OpenZaurus ROM. ** Archos PMA430. ** Siemens Simpad with Opensimpad 0.9.0/Opie ROM. ** Nokia [http://europe.nokia.com/770 770]/[http://europe.nokia.com/N800 N800]/[http://europe.nokia.com/N810 N810] [[Nokia Internet tablet]]s ([http://maemo.org maemo]). ** [[Pepper Pad]] 3. ** Motorola E680i/A780 smartphones. ** [[iLiad]] ** [[Digital Reader 1000]] and [[Digital Reader DR800SG]] ** [[Hanlin eReader]] ** ''Openinkpot'' - OS replacement for [[Hanlin eReader]] and ''Hanvon N516'' ** [[PocketBook International|Pocketbook]] - [[E Ink]] [[e-book reader]]s ** [[SmartQ 5]] and ''SmartQ 7'' ==Nikolay Pultsin== :* [http://www.geometerplus.com/ Geometer Plus LLC] is developer of [https://fbreader.org FBReader(.org)] :*[https://fbreader.org/content/about-us 2005 FBReader by Nikolay Pultsin.
